# 象棋软件包

## 软件说明

本象棋软件内置了 Pikafish 引擎，提供强大的棋力支持。由于软件包过大，无法直接上传到 Git，因此被压缩成两个分卷。

- **chess.zip.002** - [下载链接](https://example.com/chess.zip.002)

## MD5 校验

为确保文件完整性，请使用以下 MD5 值进行校验：
- `65d0b535be7fa3907b4d7125a80ddde7` - chess.zip.001
- `7922c37fc60a5ceb234f64af38ff938b` - chess.zip.002

## 使用方法

1. 下载上述两个分卷文件到同一目录
2. 使用支持分卷压缩的解压软件（如 WinRAR、7-Zip 等）解压
3. 解压后得到完整的象棋软件
4. 运行软件即可开始使用

## 功能特性

- **内置 Pikafish 引擎**：提供强大的 AI 对战能力
- **多种难度级别**：适合不同水平的棋手
- **棋谱记录**：支持保存和加载棋谱
- **界面友好**：操作简单，界面美观


# -- 遗嘱 --

```
1. 此遗嘱所立 非尔轻生 此生大志 百岁嫌少 安名立身 以安尔志
2. 去后即当火化 身不染俗物 随身贵重物品当取走利用 着随身衣物焚化即可不必另备丧服 望生者敬
3. 尔之骨灰 不可入土 随风随水撒之 不可扰于众生 有余力着撒于山巅 随马桶冲水也可 亲者勿念 恨着唾之
4. 生至今也无大财 无有可恋 若有余财由生者处决 莫不可争执 若有争执半毫不留捐了修路
5. 尔不立碑 不举任何丧仪 身后事一应从简至简 若火花无财还劳烦各位相凑 谢
6. 若重病弥留之时 可不予医 医则尽保守 不必开膛破脑之术 
   医疗之资有损生者生存 不必医 医疗之后有损本人生命质量 不必医 早解尘世之所劳 莫不可为生者增负
7. 此遗嘱立于未名笔记之二  尔小小之愿 望身生者随愿
                      ---- 未名道人  初稿于辛卯年夏 定稿于己亥年春-惊蛰
```

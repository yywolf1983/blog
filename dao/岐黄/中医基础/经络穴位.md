## 第二部分：经络穴位

### 一、经络系统总论

经络是人体内运行气血、联络脏腑肢节、沟通内外上下的通路系统，是中医理论的核心组成部分。如果把人体比作一个城市，经络就是城市的道路网——十二正经是主要干道，奇经八脉是环城高速，络脉是小街小巷，气血就像在城市中行驶的车辆，将营养物质输送到每个角落。

#### （一）经络系统的层次结构

经络系统
├─ 经脉（主干道）
│   ├─ 十二正经（12条主干线）
│   │   ├─ 手三阴经：肺经、心包经、心经
│   │   ├─ 手三阳经：大肠经、三焦经、小肠经
│   │   ├─ 足三阳经：胃经、胆经、膀胱经
│   │   └─ 足三阴经：脾经、肝经、肾经
│   │
│   └─ 奇经八脉（8条特殊干线）
│       ├─ 任脉（前正中线）
│       ├─ 督脉（后正中线）
│       ├─ 冲脉（气血之海）
│       ├─ 带脉（腰间环带）
│       ├─ 阴维脉、阳维脉（维系联络）
│       └─ 阴蹻脉、阳蹻脉（司运动）
│
├─ 络脉（分支道路）
│   ├─ 十五别络（主要分支）
│   ├─ 孙络（细小分支）
│   └─ 浮络（表浅分支）
│
├─ 经筋（筋肉系统）
├─ 皮部（皮肤分区）
└─ 经别（深入支脉）

#### （二）十二经脉的三大特点

 1. 有固定路线：如公交线路，每条经有明确走向
 2. 有特定站点：穴位就是经气停留汇聚的“公交站”
 3. 有运行时间：气血在不同时辰流注不同经脉（子午流注）

### 二、十二正经详解

#### 手三阴经（从胸走手）

比喻：肺经就像人体的“供水总管”，从胸部（中焦）出发，将水谷精微（营养物质）像水泵一样输送到体表。

### 七、经络穴位临床应用要点

#### （一）选穴原则

 1. 近部选穴：在病变局部或邻近选穴（如头痛选百会、太阳）
 2. 远部选穴：在病变所属经络远端选穴（如胃痛选足三里）
 3. 辨证选穴：根据证候选穴（如肝阳上亢选太冲）
 4. 对症选穴：针对特定症状选穴（如发热选大椎）

#### （二）配穴方法

 1. 本经配穴：同一经络穴位配合（如胃痛选中脘、足三里）
 2. 表里经配穴：表里两经穴位配合（如咳嗽选肺经太渊、大肠经合谷）
 3. 上下配穴：上肢与下肢穴位配合（如失眠选上肢神门、下肢三阴交）
 4. 前后配穴：胸腹与背腰穴位配合（如胃痛选腹部中脘、背部胃俞）
 5. 左右配穴：左右两侧同名穴位配合

#### （三）特定穴应用

 1. 五输穴：按“井主心下满，荥主身热，输主体重节痛，经主喘咳寒热，合主逆气而泄”应用
 2. 原络配穴：先病经脉的原穴与相表里经脉的络穴配合
 3. 俞募配穴：背俞穴与募穴配合（如肺病选肺俞、中府）
 4. 郄穴：多用于治疗急性病、痛症
 5. 八会穴：治疗相关组织器官疾病（如筋病选阳陵泉）
 6. 八脉交会穴：两穴一组配合使用

#### （四）操作注意事项

 1. 定位准确：骨度分寸、体表标志、手指同身寸结合
 2. 针刺深度：根据部位、体质、病情决定
 3. 刺激强度：虚证用补法，实证用泻法
 4. 治疗时间：一般留针20-30分钟

第二部分总结：
经络穴位是中医外治法（针灸、推拿、刮痧、拔罐等）的理论基础。十二正经如12条主干道，奇经八脉如8条特殊通道，穴位如道路上的站点。学习时要掌握每条经脉的循行路线、主要穴位、主治特点。重点记忆五输穴、原穴、络穴、郄穴、背俞穴、募穴、八会穴、八脉交会穴等特定穴，这些穴位临床使用频率高，疗效显著。
